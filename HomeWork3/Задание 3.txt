В файле script.js находится массив пользователей, зарагестрированных на сайте. Для каждого пользователя хранится следующая информация: id, имя, роль

1) Проверка прав пользователя
	1) Создайте функцию isAdmin
	2) Эта функция должна будет получать аргументом число, это число является индентификационным номером пользователя
	3) Данная функция должна проверить массив, найти в нём пользователя с соответствующим id и вернуть значение true, если его роль Администратор, иначе false
	4) Если пользователя с таким id не нашлось в массиве - это тоже результат false

2) Список имён
	1) Создайте функцию getNames
	2) Функция будет возвращать новый массив, который будет содержать просто список имён пользователей

3*) Разделить по ролям
	1) Создайте функцию splitUsersByRole
	2) Эта функция должна пройтись по массиву пользователей, чтобы разделить их по ролям
	3) В результате работы эта функция должна вернуть двумерный массив, который будет содержать в себе два массива, содержащие:
		1) id пользователей с ролью user
		2) id пользователей с ролью admin
	4*) Сделайте функцию более универсальной, чтобы она делила пользователей не по заранее заданным двум ролям, а для любого количества возможных вариаций ролей.
		Теперь функция должна возвращать не двумерный массив, а объект:
			Ключами этого объекта будет значение роли пользователя
			Значениями массив с id пользователей с соответствующей ролью
